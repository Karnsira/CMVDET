.. _help:

Getting Help
==================


We appreciate your feedback and questions! You can report any error or suggestion related to CMVDET as an `issue on github <https://github.com/Karnsira/CMVDET/issues>`_. If you have questions related to the manuscript, data, or any general comment or suggestion please use the `Discussions <https://github.com/Karnsira/CMVDET/discussions>`_.

Thank you!
